# Lessons — TPoTHP AI Anime Workflow

Use this file to capture corrections, pitfalls, and workflow improvements.

## Active Lessons

- Keep generation outputs inside this project root unless explicitly told otherwise.
- Keep every generated asset paired with its prompt and backend metadata.
- For still images, prefer local Codex GPT Image unless the user explicitly requests RunComfy GPT Image 2.
- For Seedance/video generation, RunComfy auth must be active before execution.
- Never expose auth tokens in logs, prompts, manifests, or reports.
